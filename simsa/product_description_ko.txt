Korean Word Lookup - 한국어 번역기/사전 (호버 + 선택 번역)

웹에서 한국어(한글)를 읽을 때 바로 뜻을 보고 싶다면 이 확장프로그램 하나면 돼.
단어 위에 마우스를 올리면(호버) 빠르게 뜻을 보고, 문장은 드래그로 선택해서 한 번에 번역할 수 있어.

주요 기능

호버 번역(단어 조회)
- 한국어 단어에 마우스를 올리면 영어 뜻/번역을 바로 표시
- Google Translate(빠름) / KRDICT(자세한 사전식 정의) 중 선택
- 깔끔한 툴팁 UI + (옵션) 한국어 정의 표시

선택 번역(문장 번역)
- 여러 단어/문장을 드래그로 선택해서 한 번에 번역
- 팝업에서 켜기/끄기 또는 `Alt+S` 단축키로 토글
- 선택 번역은 Google Translate를 사용해 자연스러운 문장 번역 제공

단축키 & 설정
- `Alt+T`: 전체 번역 on/off
- `Alt+G`: Google Translate로 전환
- `Alt+K`: Korean Dictionary(KRDICT)로 전환
- `Alt+S`: 선택 번역 on/off
- 호버 딜레이, 컴팩트 모드, 최대 정의 개수 등 커스터마이즈

개인화
- 설정값은 Chrome 스토리지(동기화/로컬)에 저장돼서 다시 켤 때 자동으로 복원
- (선택) KRDICT API 키를 팝업에서 직접 입력해서 기본 키를 대체 가능

개인정보/권한 안내
- 번역/정의 조회를 위해, 사용자가 호버/선택한 텍스트가 Google Translate 또는 KRDICT API로 전송될 수 있어.
