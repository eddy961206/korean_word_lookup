Korean Word Lookup - 한국어 학습자를 위한 빠른 단어/문장 번역

한국어 웹페이지를 읽다가 흐름이 끊기는 순간을 줄여줘.

Korean Word Lookup은 한국어를 공부하는 사용자를 위한 Chrome 확장 프로그램이야. 한국어 단어 위에 마우스를 올리면 영어 뜻을 빠르게 보고, 문장을 드래그로 선택하면 한 번에 번역할 수 있어.

이런 사람에게 좋아
- 한국어 기사, 게시글, 문서, 학습 자료를 읽는 한국어 학습자
- 매번 새 탭을 열지 않고 바로 영어 뜻을 보고 싶은 사람
- Chrome, Windows, Mac, Chromebook에서 매일 한국어 읽기 연습을 하는 사람

주요 기능

단어 호버 조회
- 한국어 단어에 마우스를 올리면 영어 뜻/번역 표시
- 빠른 Google Translate와 자세한 KRDICT Korean Dictionary 중 선택
- 선택한 소스에서 결과가 없으면 다른 소스를 자동으로 시도하는 Auto-Fallback 지원
- 깔끔한 툴팁과 선택 가능한 한국어 정의 표시

문장 선택 번역
- 한국어 문장이나 문단을 드래그로 선택해서 즉시 번역
- 팝업 또는 단축키로 선택 번역을 켜고 끌 수 있음
- 단어 하나의 뜻만으로 문맥이 부족할 때 유용함

설정과 단축키
- 클릭 한 번으로 번역 기능 켜기/끄기
- Windows/Linux/ChromeOS: Alt+T, Alt+G, Alt+K, Alt+S
- Mac: Option+Shift+T, Option+Shift+G, Option+Shift+K, Option+Shift+S
- 호버 지연 시간, 컴팩트 툴팁, 정의 개수 조절 가능

개인정보 안내
- 사용자가 호버하거나 선택한 텍스트만 번역/정의 조회를 위해 Google Translate 또는 KRDICT로 전송될 수 있어
- 설정값은 Chrome 저장소에 저장돼
- 제품 분석에는 호버/선택한 원문 텍스트를 보내지 않아
